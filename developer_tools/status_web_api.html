Status DApp API

On top of regular web3 access, Status offers a set of API available to DApps developers. This API follows the EIP1102 standard that will soon be used for web3 injection.

When Status is the host browser, DApp developers can request access to Status-specific data using the JavaScript API available at window.web3.currentProvider.status.

JavaScript API methods

  • getContactCode provides access to a user’s Whisper ID, which can be used to initiate chats or add a contact.
